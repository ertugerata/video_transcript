# YouTube & Ses Dosyası Özetleyici (Video Transcript & Summarizer)

Bu proje, YouTube videolarından ve yüklenen ses dosyalarından transkript çıkaran, Google Gemini API kullanarak özetleyen ve tüm verileri **Supabase** bulut veritabanında saklayan gelişmiş bir Flask web uygulamasıdır.

Uygulama **Client-Server** mimarisi ile çalışacak şekilde tasarlanmıştır:
*   **Web İstemcisi (`app.py`):** Kullanıcı arayüzünü sunar, veritabanı ile konuşur ve işlemleri başlatır.
*   **MCP Medya Sunucusu (`mcp-media-server/`):** Ağır medya işlemlerini (Whisper, YouTube indirme) yöneten arka uç bileşenidir.

> **Detaylı Sunucu Dokümantasyonu:** Medya işleme modülü, Whisper modelleri ve sunucu kurulumu hakkında detaylı bilgi için [**mcp-media-server/README.md**](mcp-media-server/README.md) dosyasını inceleyin.

## Özellikler

*   **YouTube Desteği:** Video linki üzerinden transkript ve özet alma.
*   **Yapay Zeka Özeti:** Google Gemini (Pro/Flash) ile otomatik özetleme.
*   **Veritabanı Kaydı:** Supabase entegrasyonu ile geçmiş sorguları saklama ve önbellekleme.
*   **Dışa Aktarma:** Transkriptleri `.txt` veya `.md` olarak indirme.
*   **İstemci Odaklı:** Hafif yapısı sayesinde minimum sistem gereksinimi ile çalışır. (Ağır işlemler uzaktaki veya yerel MCP sunucusuna devredilebilir).

## Gereksinimler (İstemci)

*   **Python 3.8+**
*   **Supabase Hesabı**
*   **Google Gemini API Anahtarı**

## Kurulum

1.  **Depoyu klonlayın:**
    ```bash
    git clone <repo-url>
    cd <repo-folder>
    ```

2.  **İstemci Paketlerini Yükleyin:**
    ```bash
    pip install -r requirements.txt
    ```
    *(Not: Whisper ile yerel ses işleme yapmak istiyorsanız, sunucu bağımlılıklarını da yüklemeniz gerekir. Bkz: [Sunucu Kurulumu](mcp-media-server/README.md))*

3.  **Çevresel değişkenleri ayarlayın:**
    Proje ana dizininde `.env` dosyası oluşturun:
    ```env
    GEMINI_API_KEY=sizin_gemini_api_anahtariniz
    SUPABASE_URL=https://sizin-proje-id.supabase.co
    SUPABASE_KEY=sizin_supabase_anon_key
    ```

4.  **Supabase Veritabanı Kurulumu:**
    Supabase SQL Editor'de aşağıdaki tabloyu oluşturun:
    ```sql
    create table transcripts (
      id bigint generated by default as identity primary key,
      video_id text,
      url text,
      full_transcript text,
      simple_transcript text,
      language text,
      summary text,
      created timestamptz default now()
    );
    ```

## Kullanım

Uygulamayı başlatmak için:

```bash
python app.py
```

Tarayıcınızda **`http://localhost:5000`** adresine gidin.

*   YouTube URL'sini girerek transkript ve özet alabilirsiniz.
*   Geçmiş aramalarınızı görüntüleyebilir ve indirebilirsiniz.
*   Yerel dosya işlemleri için uygulamanın bir MCP sunucusuna erişimi olması veya yerel modüllerin yüklü olması gerekir.

## Proje Yapısı

*   `app.py`: Ana web arayüzü.
*   `mcp_client_utils.py`: MCP sunucusu ile iletişim modülü.
*   `mcp-media-server/`: Arka uç medya işleme sunucusu ve modülleri.
